CLASSIFIER MODELS

Classifier models are used to categorize data into predefined classes or labels. These models learn from labeled datasets, where the target (output) is a categorical variable (e.g., "spam" vs "not spam", "setosa", "versicolor", "virginica"). Examples include K-Nearest Neighbors, Decision Trees, and Logistic Regression.

Input Requirements:

  • A CSV file with at least one target column (categorical).
  • All input features should be numerical (non-numerical must be preprocessed)
  • The name of the target column must be specified in the form.

REGRESSION MODELS

Regression models are used to predict continuous values. These models learn from labeled datasets where the target is a numerical variable (e.g., price, temperature, age). Common models include Linear Regression, Decision Tree Regressor, and Random Forest Regressor.

Input Requirements:

  • A CSV file with a numeric target column.
  • All input features should be numerical and without missing values.
  • The target column must be selected when training.

CLUSTERING MODELS

Clustering models are used for unsupervised learning, meaning they group similar data points together without needing a target label. They are useful for data exploration, pattern discovery, and segmentation. Common algorithms include K-Means and DBSCAN.
