CLUSTERING MODELS

Clustering models are used for unsupervised learning, meaning they group similar data points together without needing a target label. They are useful for data exploration, pattern discovery, and segmentation. Common algorithms include K-Means and DBSCAN.

Input Requirements:

  • A CSV file without a target column.
  • All input features should be numerical.
  • The number of clusters may be predefined (e.g., for K-Means) or determined automatically (e.g., by DBSCAN).
